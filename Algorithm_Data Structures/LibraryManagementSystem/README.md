Exercise 6: Library Management System
1. Understand Search Algorithms
Linear Search

Linear search checks each element one by one until the desired element is found or the list ends.

Algorithm:

Start from the first element.
Compare the title with the search key.
If matched, return the book.
Otherwise, move to the next element.
Repeat until found or the list ends.

Advantages

Simple to implement.
Works on both sorted and unsorted data.

Disadvantages

Slow for large datasets.
Binary Search

Binary search works only on sorted data. It repeatedly divides the search space into two halves.

Algorithm

Find the middle element.
Compare it with the search key.
If equal, return the book.
If the key is smaller, search the left half.
Otherwise, search the right half.
Repeat until found or the search space is empty.

Advantages

Very fast for large sorted datasets.

Disadvantages

Requires the array to be sorted.
